Vidyacraft Season 5 Setup Instructions

First thing's first, you can use a completely vanilla (default) version of minecraft but the intended way is to use one of the following modpacks:
  • VG-S5: The standard way to play (choose this one if you're unsure)
  • VG-S5 (optifine): Same as VG-S5, just using optifine instead of iris for shaders. Only use this pack if you are sure you'll need it as it runs slower.
